# Lingo

A chat application

## Features 

- Direct messaging

## Technologies

- React: UI library
- Socket.io: For bidirectional communication
- Chakra UI: Component library

## Usage

First get the server running:

```bash
cd lingo-server
npm i
npm run dev
```

Then get the application running:

```bash
cd lingo-app
npm i
npm run dev
```

It is a vite application possibly running on `http://localhost:5173`.
